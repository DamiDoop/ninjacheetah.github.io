NCX-Installer

GitHub Page

Available through NCX-Installer: Technically Yes

NCX-Installer was created as an easy way to install and update all of the Visual Studio projects I make. It is also a good way for me to test checking for updates and auto-updating. It currently semi-automatically updates by downloading the latest installer version from the GitHub page. It has the ability to download the latest .zip releases of my progams, or automatically download and run the .msi for programs that support installation and are not portable only. The best part is it only needs to update when I make a new program, because it always grabs the latest release from GitHub.
